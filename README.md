# CodeAlpha_Phishing_Awareness_Training
A cybersecurity awareness project developed for the CodeAlpha Cyber Security Internship. This project educates users about phishing attacks, fake websites, social engineering techniques, email security, and preventive measures through an interactive presentation, real-world examples, and quizzes.
# Phishing Awareness Training

## Overview

Phishing is one of the most common cyberattacks used to steal sensitive information such as usernames, passwords, banking details, and personal data. This project was developed as part of the **CodeAlpha Cyber Security Internship** to educate users on recognizing phishing attempts, understanding social engineering tactics, and adopting safe online practices.

The presentation includes real-world phishing examples, methods used by attackers, preventive techniques, and interactive quizzes to improve cybersecurity awareness.

---

## Objectives

- Understand what phishing is and why it is dangerous.
- Learn different types of phishing attacks.
- Recognize phishing emails and fake websites.
- Understand common social engineering techniques.
- Follow cybersecurity best practices to prevent phishing attacks.
- Improve awareness through quizzes and practical examples.
